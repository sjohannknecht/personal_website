Action Unit Erkennung

Action Units sind ein Konzept aus der Psychologie zur Beschreibung der Aktivierung von Gesichtsmuskeln. Damit lassen sich zum Beispiel Gesichtsausdrücke oder Emotionen beschreiben.

Aktivierungen von Action Units können mithilfe von Modellen maschinellen Lernens in Videos erfasst werden. In meiner Bachelor-Arbeit habe ich verschiedene bereits verfügbare Modelle zur Action Unit Erkennung bezüglich ihrer Verwendung im Browser evaluiert.

Im Zuge dessen habe ich eine kleine JavaScript-Bibliothek zur Action Unit Erkennung geschrieben und diese in einer Web App mit React demonstriert. Das Projekt ist mit dem Create React App Framework erstellt.

In der Web App kann man sich mit einer Webcam aufnehmen und eine Action Unit Erkennung durchführen lassen. Dies geschieht komplett im Browser, ohne dass dabei Bilddaten an den Webserver gesendet werden.

Details zur Implementierung und wie es mit dem Projekt weitergeht, werden im GitHub Repo zu finden sein, sobald es veröffentlicht ist.
